That sentance was size h1. These two are size h2.

This line is an example of size h3. The next line will be h4.

You have probably noticed that as the h value increases the text size gets smaller.

Of course the h system can do a lot more then just change the size of the text.
With CSS you can make each h use a different font, color, size, postioning or just about anything you want.
